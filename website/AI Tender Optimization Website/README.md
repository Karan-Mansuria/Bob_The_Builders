
  # AI Tender Optimization Website

  This is a code bundle for AI Tender Optimization Website. The original project is available at https://www.figma.com/design/fYCCsiFk5OP8sFfcsj4TTt/AI-Tender-Optimization-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  To connect to the backend API, set `VITE_API_BASE` (defaults to `http://localhost:8000`). Example:

  ```bash
  VITE_API_BASE=http://localhost:8000 npm run dev
  ```
